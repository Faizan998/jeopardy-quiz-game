# Sound Files for Jeopardy Quiz Game

This directory contains sound files used in the Jeopardy Quiz Game.

## Required Sound Files

The application looks for the following sound files:

1. `click.mp3` - Played when a user clicks on a question
2. `select.mp3` - Played when a user selects an answer option
3. `correct.mp3` - Played when a user submits a correct answer
4. `incorrect.mp3` - Played when a user submits an incorrect answer

## Adding Sound Files

You can add your own sound files by placing MP3 files with the names above in this directory.

If the sound files are not present, the application will continue to function normally without sound effects.
